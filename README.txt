Cats' Sight

Decription:

This project aims to practice HTML,CSS and JavaScript.

The three representative tasks of Cat’s Sight are: 
1.Browsing. Users can read the information I offered in Cats' Sight.  
2.Post something. Users can post something by clicking "Post" in the left menu after logging-in.
3.Reply message. After logging in, users can click ME in main page, then select message in the bottom of navigation bar to choose which message they want to reply.
